# Text Autocomplete with LSTM and Transformers Project

В этом проекте реализована модель автодополнения текста — нейросеть, которая по началу фразы предсказывает её продолжение. Задача формулируется как последовательное предсказание следующего токена до достижения конца текста или заданной длины.

На первом этапе данные были очищены, нормализованы и токенизированы. Затем из текстов сформированы обучающие примеры в формате «вход → следующий токен», а датасет разбит на train/validation/test. В качестве датасета использован сет коротких постов-твиттов sentiment140:
wget -P ./data/ https://code.s3.yandex.net/deep-learning/tweets.txt

Основная модель проекта — **рекуррентная нейронная сеть на базе LSTM**. Она обучается предсказывать следующий токен и умеет генерировать текст по заданному префиксу. Для оценки качества используются функции потерь и метрика ROUGE (ROUGE-1 и ROUGE-2), отражающая качество сгенерированного текста.

Дополнительно в проекте используется **предобученная трансформерная модель DistilGPT2**. Она применяется для той же задачи автодополнения, что позволяет сравнить качество и поведение лёгкой LSTM-модели и более мощного трансформера.

В рамках экспериментов анализируется компромисс между качеством генерации и вычислительными затратами, что важно для применения моделей на устройствах с ограниченными ресурсами, таких как смартфоны.
